# JobInboxAgent v2

A web app that monitors your Gmail 24/7 and alerts you on Telegram the moment an important job email arrives — interview invites, offers, or rejections.

No more missing the email that matters.

## How it works
1. Login with your Google account
2. Enter your Telegram bot token and chat ID
3. Done — AI monitors your inbox every 5 minutes automatically

## What gets alerted
- Interview invites and next steps
- Job offers
- Rejections
- Everything else is silently ignored

## Built with
- Python + Flask
- Gmail API + Google OAuth
- Groq AI (LLaMA 3.3 70B)
- Few-shot prompting
- Telegram Bot API
